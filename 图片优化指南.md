# 图片压缩优化指南

## 当前问题
最大图片 249KB，对移动端太慢。

## 推荐方案

### 方案1：使用在线工具压缩（推荐）
1. 访问 https://tinypng.com/ 或 https://squoosh.app/
2. 上传以下大图片进行压缩：
   - art3_004.jpg (249KB)
   - art2_004.jpg (229KB)  
   - art1_006.jpg (213KB)
   - starry-nailong-bg.jpg (208KB)
   - art1_004.jpg (207KB)
3. 下载压缩后的图片替换原文件
4. 通常可压缩 60-80%

### 方案2：使用 Python 批量压缩
```bash
pip install Pillow
python compress_images.py
```

### 方案3：使用 WebP 格式
将图片转换为 WebP 格式，体积减少 30-50%

## 目标
- 单张图片 < 100KB
- 背景图 < 50KB
- 总图片资源 < 2MB
